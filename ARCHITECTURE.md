## Request Flow

### Web Dashboard Request
```
Client Browser
  в”Ӯ
  в”ңв”Җ GET http://localhost:5000/
  в”Ӯ  в””в”Җв–ә Flask dashboard.py вҶ’ dashboard_html()
  в”Ӯ
  в”ңв”Җ GET /api/config
  в”Ӯ  в””в”Җв–ә config.py вҶ’ load filters
  в”Ӯ
  в”ңв”Җ POST /api/config
  в”Ӯ  в””в”Җв–ә config.py вҶ’ save filters
  в”Ӯ
  в”ңв”Җ GET /api/stats
  в”Ӯ  в””в”Җв–ә database.py вҶ’ get_today_stats()
  в”Ӯ
  в”ңв”Җ POST /api/agent/start
  в”Ӯ  в””в”Җв–ә scheduler.py вҶ’ start()
  в”Ӯ
  в””в”Җ GET /api/agent/status
     в””в”Җв–ә scheduler.py вҶ’ get_status()
```

### Job Application Flow
```
Scheduled Trigger
  в”Ӯ
  в”ңв”Җ dice_scraper.scrape_jobs()
  в”Ӯ  в””в”Җ Playwright вҶ’ Dice.com вҶ’ HTML parsing
  в”Ӯ
  в”ңв”Җ job_matcher.filter_jobs()
  в”Ӯ  в””в”Җ Apply all filters
  в”Ӯ
  в”ңв”Җ applicant.apply_to_jobs()
  в”Ӯ  в”ңв”Җ Playwright login
  в”Ӯ  в”ңв”Җ For each job:
  в”Ӯ  в”Ӯ  в”ңв”Җ Navigate to URL
  в”Ӯ  в”Ӯ  в”ңв”Җ Click apply
  в”Ӯ  в”Ӯ  в””в”Җ Confirm
  в”Ӯ  в””в”Җ database.add_application()
  в”Ӯ
  в””в”Җ database.update_daily_stats()
     в””в”Җ SQLite update
```

## Error Handling Flow

```
Any Error Occurs
  в”Ӯ
  в”ңв”Җв–ә logger.error() вҶ’ logs/agent_*.log
  в”ңв”Җв–ә Continue execution
  в”ңв”Җв–ә Retry if applicable
  в””в”Җв–ә Update stats
```

## Performance Metrics

```
Resource Usage:
в”ңв”Җ CPU: 5-10% (during cycle), <1% (idle)
в”ңв”Җ Memory: 200MB (Chromium), 50MB (app)
в”ңв”Җ Disk: 10-20MB per 10K jobs
в””в”Җ Network: Minimal (one request/interval)

Timing:
в”ңв”Җ Job scrape: 5-10 seconds
в”ңв”Җ Filtering: <1 second
в”ңв”Җ Applications: 30 jobs/hour (with delays)
в”ңв”Җ Database ops: <100ms
в””в”Җ Full cycle: 10-30 minutes
```
